Immutable - Values that cannot change
-----------------------------
Tuple 
Definition - An immutable list 
Attributes 
- Uses () instead of [].
- You do not need to explicitly define
EX Both work: tuple(x) , (x)
-----------------------------
Code Styling PEP8
- four spaces per indentation
- 79 character limit for readability for code
- Limit comments to 72 characters per line
Dictionaries
-----------------------
- Uses {}
- Retain the order in which they were defined
- When printed or looped through will retain the order in which they were added
syntax: dict_name = {}
